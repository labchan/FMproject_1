Application Title : Facility Management - Repair Order Application

The Objective of this application is to facilitate users to record / maintain the repair order record

In this application, it is using Sqlite database for the record storage

and It is using the git in local computer and github for the version control.


The application have basically 4 functions:

1) create Repair Order
2) display repair order record and select & view particular repair order
3) edit and update a particular repair order
4) delete a particular repair order
